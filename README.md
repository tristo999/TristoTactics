# TristoTactics

A tactical strategy game built with Godot 4.

## Features
- Turn-based tactical gameplay
- Modular, scalable Godot project structure
- Main menu and settings UI
- Keyboard and mouse controls
- Easily extensible for new levels, characters, and features

## Getting Started
1. **Clone the repository:**
   ```
   git clone https://github.com/tristo999/TristoTactics.git
   ```
2. **Open in Godot 4:**
   - Launch Godot 4.
   - Select the `project.godot` file in this folder.
3. **Run the game:**
   - Press F5 or click the Play button in the Godot editor.
   - The game starts at the main menu.

## Folder Structure
- `scenes/` — Game scenes (levels, UI, characters)
- `scripts/` — GDScript files for game logic
- `assets/` — Art, tilesets, and other resources
- `addons/` — Godot plugins

## Controls
- **Arrow keys / WASD:** Move camera
- **Mouse:** Interact with UI
- **Start Game:** Begin playing
- **Settings:** Adjust volume
- **Quit:** Exit the game
